## Notas de uso

1. Pega el prompt completo (desde la línea "Necesito que generes…" hasta "Sin marcas de agua.") en ChatGPT, Claude o tu IA preferida.
2. Si el tool no genera PDFs directos, pídele que arme el contenido en un canvas con maquetación HTML/Markdown y lo exporte como PDF.
3. Para las imágenes de producto puedes usar una segunda llamada a ChatGPT/DALL·E con el bloque "IMÁGENES DE PRODUCTOS" como prompt visual.
4. Los precios mayoristas son los oficiales que entregó el cliente. Para el agua bacteriostática se calcularon al 77% del público (ratio histórico).
5. Revisa siempre las tablas antes de imprimir — los descuentos por volumen y métodos de pago están como placeholder para que tú los definas.
